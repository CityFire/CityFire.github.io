# 前置基础知识

确保你掌握一些基本 Git 和 Python 的基础知识。 第一部分:Git 基础

## 1. 什么是Git? 

Git是一个分布式版本控制系统，用于跟踪文件的更改和协作开发。它就像是一个项目的时光机，可以:

- 记录文件的所有更改历史
- 允许多人协作开发
- 可以随时回到项目的任何历史版本

![Git 可以通过中心化的方式允许多端，多人协同合作编程](images/ai_pratice_git.png)

## 2. Git的基本概念

![Git 记录文件修改历史方式，可以实现精准的版本管理](images/ai_record_git.png)

1. **工作区(WorkingDirectory)**
- 这是你实际编辑文件的地方
- 可以理解为你能在电脑上直接看到的项目文件夹

2. **暂存区(StagingArea)**
- 临时存储你想要提交的改动
- 使用 git add 命令将改动添加到暂存区

3. **本地仓库(LocalRepository)**
- 存储项目的所有版本信息
- 使用 `git commit` 命令将暂存区的改动提交到本地仓库

## 3. 基本Git命令

```
1 # 初始化Git仓库 
2 git init
3
4 # 查看文件状态 5 git status
6
7 # 添加文件到暂存区
8 git add 文件名
9 git add . # 添加所有文件
10
11 # 提交更改
12 git commit -m "提交说明" 
13
14 # 查看提交历史
15 git log
```

# 进阶应用：股价预测

## 背景介绍

![](images/ai_pratice_stock.png)

1. **为什么用Python预测股价?**
- Python具有强大的数据处理能力，特别是通过Pandas和NumPy等库
- 拥有丰富的机器学习工具，如Scikit-learn，支持多种预测算法 
- 优秀的可视化能力，可以直观展示分析结果

2. **预测股价的意义:**
- 为投资决策提供数据支持，减少主观判断带来的⻛险 
- 帮助识别市场⻛险，提供预警机制 
- 验证交易策略的有效性，发现市场规律 
- 作为教学实践的优秀案例，帮助学习数据分析和机器学习

3. **技术路线特点:**
- 采用随机森林算法，具有良好的抗噪声能力和泛化能力 
- 综合考虑技术指标、市场情绪等多维度特征 
- 完整的数据处理流程和模型评估体系

4. **应用价值:**
- 教学实践:帮助学习数据分析和机器学习 
- 研究参考:支持市场效率研究和策略验证
- 投资参考:辅助趋势分析和⻛险评估

5. **注意事项:**
- 预测结果仅供参考，需要结合其他因素综合判断 
- 市场存在不确定性，任何预测方法都有局限性 
- 重视⻛险控制，避免过度依赖预测结果

这个项目不仅是一个股票预测工具，更是一个优秀的学习案例，帮助学习者理解数据分析、机器学习在金融领域的 应用。通过这个项目，学习者可以掌握:

- 数据获取和预处理技术 
- 特征工程方法
- 机器学习模型训练和评估 
- 数据可视化技巧

需要特别强调的是，**这个系统的主要目的是教学和学习，帮助理解Python在金融数据分析中的应用，而不是作为实 际投资的决策工具。**
