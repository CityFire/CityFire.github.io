## 🚀 快速开始

### 本地预览

```bash
# 克隆仓库
git clone git@github.com:CityFire/CityFire.github.io.git
cd CityFire.github.io

# 安装依赖
npm install

# 启动本地服务器
hexo server

# 访问 http://localhost:4000
```

### 新建文章

```bash
# 创建新文章
hexo new post "文章标题"

# 生成静态文件
hexo generate

# 部署到 GitHub Pages
hexo deploy
```
